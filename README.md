Todo App
A simple Todo App built using HTML, CSS, and JavaScript that allows users to add and delete tasks.

Features:
Add new tasks to the list.
Delete tasks from the list.
Simple and intuitive user interface.
Built with event delegation for efficient deletion handling.

Technologies Used:
HTML: For the structure of the page.
CSS: For styling and layout of the app.
JavaScript: For adding task functionality and handling events.

How It Works:
Add Task: Type your task into the input field and click the "Add Task" button. The task will be added to the list below.
Delete Task: Click the "delete" button next to any task to remove it from the list.

Code Structure:
index.html: Contains the HTML structure of the app.
style.css: The styles for the Todo App, defining the layout, colors, and interactions.
app.js: Contains the JavaScript logic for adding tasks and handling the deletion of tasks.

Usage:
After adding a task, you can click the "delete" button next to any task to remove it from the list.
The app uses event delegation to handle the deletion of tasks, so it works efficiently as new tasks are added.
